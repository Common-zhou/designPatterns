# 什么叫单例
    单例就是让整个虚拟机内存中只有一个该类的对象,比如一些连接类,工具类。这样可以节约内存

# 单例实现模式
## 1.直接生成(见Singleton1)
```
    饿汉式
    类加载到内存后，就实例化一个单例，JVM保证线程安全
    简单实用，推荐使用！
    唯一缺点：不管用到与否，类装载时就完成实例化
    Class.forName("")
    （话说你不用的，你装载它干啥）
```

## 2.懒加载(见Singleton2)
```
    lazy loading
    也称懒汉式
    虽然达到了按需初始化的目的，但却带来线程不安全的问题
    可以通过synchronized解决，但也带来效率下降
```
